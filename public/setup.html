3. Redeploy (or it'll auto-pick it up on next start)
The token is also saved to file for immediate use without restarting.
⚔️ Mahabharat Channel
Railway Variable: MAHABHARAT_YOUTUBE_REFRESH_TOKEN
Not connected
Connect a separate YouTube channel for Mahabharat Shorts. If not set, falls back to Sameta channel.
📋 Refresh Token — copy this into Railway Variables
1. Go to Railway → Your Project → Variables
2. Set MAHABHARAT_YOUTUBE_REFRESH_TOKEN = the value above
3. Redeploy (or it'll auto-pick it up on next start)
The token is also saved to file for immediate use without restarting.

Tokens are stored in Railway's filesystem (output/) and auto-loaded on startup. They survive process restarts but are wiped on redeploy — that's why you should also save to Railway Variables.
